# 아뜨랑스 쇼핑몰 포트폴리오 실행 가이드 (UTF-8)

## 0. 프로젝트 구성
- 백엔드: `shop` (Spring Boot 3 + PostgreSQL + Spring Security)
- 프론트엔드: `pure` (Vue 3 + Vite)
- DB 스크립트: `attrangs_ddl1.sql`, `attrangs_ddl2.sql`

## 1. DB 생성 및 쿼리 실행 순서

### 1-1. PostgreSQL 서비스 실행
- Windows 예시(관리자 PowerShell):
```powershell
net start postgresql-x64-14
```

### 1-2. DB 생성
```sql
CREATE DATABASE shop_attrangs;
```

### 1-3. 스크립트 실행 (순서 고정)
```bash
psql -U postgres -d shop_attrangs -f attrangs_ddl1.sql
psql -U postgres -d shop_attrangs -f attrangs_ddl2.sql
```

### 1-4. 참고
- `attrangs_ddl1.sql`: 핵심 테이블 + 카테고리(OUTER, TOP, PANTS, SHOES, BAG, ACC, OUTLET) + 기본 샘플 데이터
- `attrangs_ddl2.sql`: 운영 보강 ALTER + 리뷰/Q&A 데모 데이터

## 2. IntelliJ 백엔드 설정 및 실행 (shop)

### 2-1. 프로젝트 열기
1. IntelliJ IDEA 실행
2. `shop_musinsa/shop` 폴더 Open
3. Gradle Sync 완료 대기

### 2-2. JDK 설정
- `Project SDK`: Java 21
- Gradle JVM도 Java 21로 맞춤

### 2-3. application.yml 확인
파일: `shop/src/main/resources/application.yml`
```yaml
spring:
  datasource:
    url: jdbc:postgresql://localhost:5432/shop_attrangs
    username: postgres
    password: 1004
server:
  port: 8085
```
- PostgreSQL 계정 비밀번호와 다르면 `password` 값을 실제 값으로 수정

### 2-4. 백엔드 실행
1. `ShopApplication` 실행
2. 정상 기동 확인: `http://localhost:8085`

### 2-5. 관리자 계정
- 이메일: `admin@attrangs.com`
- 비밀번호: `admin1234`

## 3. Vue 프론트 실행 (pure)

### 3-1. 의존성 설치
```bash
cd pure
npm install
```

### 3-2. 개발 서버 실행
```bash
npm run dev
```

### 3-3. 접속
- 프론트: `http://localhost:5173`
- Vite Proxy가 `/v1/*` 요청을 `http://localhost:8085`로 전달

## 4. 구현 체크포인트

### 4-1. 백엔드
- Spring Security

- 상품
  - 관리자 상품 CRUD + 이미지 업로드

  - 담기/조회/삭제
- 주문

  - 관리자 주문 상태 변경
